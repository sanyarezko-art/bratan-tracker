# Contributing

## Добавление в каталог

Открой PR, который правит только `catalog.json`. Структура элемента:

```json
{
  "title": "Короткий заголовок",
  "description": "Одно-два предложения о чём это (необязательно).",
  "tags": ["video", "creative-commons"],
  "size": "129 МБ",
  "magnet": "magnet:?xt=urn:btih:…&dn=…&tr=…&ws=…"
}
```

Требования:

- `magnet` обязателен, должен быть валидным magnet-URI с `xt=urn:btih:…`.
- Желательно включить как минимум один WebSocket-трекер
  (`tr=wss%3A%2F%2Ftracker.openwebtorrent.com` и т.п.) — без них WebTorrent
  в браузере не найдёт пиров.
- Проверь, что торрент реально сидирован WebRTC-пирами или есть WebSeed
  (`ws=…`) — иначе у посетителей в браузере он не скачается.
- Добавляй только то, что ты вправе распространять. Ссылки на материалы с
  нарушением авторских прав будут отклоняться и удаляться.

JSON должен быть валидным (ну правда). Если добавляешь в `items`, не забудь
запятую.

## Код

- Статика: `index.html`, `styles.css`, `app.js`.
- Ничего серверного. Не добавляй зависимости, требующие бэкенда.
- Не добавляй трекинг, аналитику, сторонние шрифты, сторонние картинки.
- Если добавляешь внешний JS — только с SRI-хэшем и на строгом CDN.

## Рекомендация для тестирования

Запусти локально `python3 -m http.server 8080` и открой
`http://localhost:8080/`. Прокинь magnet из каталога — должны появиться
файлы и начать тикать пиры.
