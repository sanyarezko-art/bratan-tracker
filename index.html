Брось файл в окно приложения — получи magnet-ссылку. Отправь её получателю. Байты идут напрямую, без нашего сервера и без логов. Бесплатно. Windows, macOS, Linux.

🪟 Windows Setup .exe · Portable 🍎 macOS .dmg · Intel + Apple Silicon 🐧 Linux .AppImage x86_64

Все сборки — на странице релизов GitHub. SHA-256 чексумы — в файле SHA256SUMS.txt рядом.

Без сервера

Файл не проходит через наш сервер — только TCP/UDP/WebRTC между двумя компьютерами.

Без аккаунтов

Ни регистрации, ни e-mail, ни телефона. Открыл, бросил файл — поделился ссылкой.

Без логов

Приложение не собирает аналитику. Что ты раздаёшь — знаешь только ты и получатель.

Большие файлы

Полный BitTorrent-стек: TCP, UDP, uTP, DHT, LSD, WebRTC. Отдаёт и принимает гигабайты.

Как работает

  1. Скачай и поставь БРАТАН на оба ПК — твой и того, кому отправляешь.
  2. Открой приложение. Перетяни файл в окно — получишь magnet-ссылку.
  3. Отправь magnet собеседнику любым способом (мессенджер, e-mail).
  4. Он вставит её в своём БРАТАНе — файл пойдёт напрямую от тебя к нему.
  5. Оставь приложение открытым, пока идёт передача.
Windows / macOS ругается «неизвестный издатель» — это нормально?

Да. Мы не покупаем сертификат Microsoft / Apple за $200–500/год — поэтому при первом запуске ОС покажет предупреждение.

Windows: в окне SmartScreen нажми «Подробнее» → «Выполнить в любом случае».

macOS: правый клик по .app → «Открыть» → подтверди один раз. Если macOS вообще не даёт — Терминал, xattr -dr com.apple.quarantine /Applications/Bratan.app.

Linux: chmod +x Bratan-*.AppImage и запускай.

Хочешь убедиться, что качаешь оригинал — посчитай SHA-256 от скачанного файла и сравни со строкой в SHA256SUMS.txt в релизе.
