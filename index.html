Торренты прямо в браузере. Без сервера. Без логов.

Это статический сайт на GitHub Pages. Всё — клиент, «трекер», передача данных — происходит в твоём браузере по WebRTC через публичные WebSocket-трекеры WebTorrent. У сайта нет бэкенда, нет аккаунтов, нет кук, нет аналитики, нечего логировать в принципе.

Прочитай перед использованием

  • Это не магическая анонимность. Как в любом торрент-клиенте, твой IP виден пирам, с которыми браузер устанавливает WebRTC-соединение. Если нужна анонимность — запускай через VPN или Tor Browser до открытия сайта.
  • Мы ничего не храним. Репозиторий держит только JSON с магнет-ссылками. Файлы идут напрямую между браузерами. У владельца сайта нет ни доступа к контенту, ни IP-логов.
  • Соблюдай закон и авторские права. Не раздавай то, на что у тебя нет прав.
  • Данные временные. Скачанные куски живут в памяти вкладки. Закрыл вкладку — раздача остановилась, кэш очистился (если только ты не сохранил файлы на диск явно).

О проекте

БРАТАН-трекер — это чисто статическая страница на GitHub Pages. Передача данных идёт peer-to-peer через WebTorrent (WebRTC). Сигналинг — публичные WebSocket-трекеры (wss://tracker.openwebtorrent.com, wss://tracker.webtorrent.dev, wss://tracker.files.fm:7073/announce).

Сайт не шлёт аналитику, не ставит куки, не грузит сторонние шрифты. Внешний код — только библиотеки WebTorrent и QR-генератор с jsdelivr, пришпиленные по Subresource Integrity.
